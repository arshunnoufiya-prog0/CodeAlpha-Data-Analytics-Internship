# CodeAlpha Data Analytics Internship Projects

This repository contains the projects completed during my CodeAlpha Data Analytics Internship.

## Task 1 – Web Scraping

* Scraped book data from BooksToScrape.com using Python.
* Extracted Title, Price, Rating, and Availability.
* Saved the data into a CSV file.

## Task 2 – Exploratory Data Analysis (EDA)

* Performed data cleaning and missing value analysis.
* Generated descriptive statistics and correlation analysis.
* Created visualizations and identified patterns in the dataset.

## Task 3 – Data Visualization

* Built charts and dashboards using Matplotlib and Seaborn.
* Created bar charts, pie charts, scatter plots, histograms, and dashboards.

## Task 4 – Sentiment Analysis

* Performed sentiment analysis on customer reviews using NLP.
* Used VADER and TextBlob for sentiment classification.
* Generated sentiment visualizations and word clouds.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* BeautifulSoup
* Requests
* NLTK
* VADER Sentiment Analyzer
* TextBlob
* WordCloud
